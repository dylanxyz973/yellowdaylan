Objectives:

Application of node.js and Express.js with EJS templating engine for rendering. CRUD for main resource. dynamic content designing part of mini supermarket website with role-based authorisation for both users and admins. Retrieve and display product information using SQL.

Functionalites:

User authentication: Registration for all new users. Log-in and log-out functionality for both users and admins. Users: Add products to cart by clicking buy products. They have the option to select number of quantity of products they want and view the product details. Admins: Update (edit) the product details, add new products to inventory, delete products from inventory with a confirmation prompt. Features: Product page display username and role. Product on Hover will turn colour to enhance attention.

Building process:

Completed within a timeline of 2 days, this is my first ever functional website I created incoporates an simple CSS yet user-friendly design and layout. Given more time, I would have make it look more realistic and added tiny feautres such prompt when items sold out, search/filter functionality. There is a feature I incoporated which admins will need a seperate user account to access the user functionalities which means admin access admin functionalities only and cannot access user functionalites unless they create user account.
