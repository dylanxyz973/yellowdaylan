Objectives:

Application of React Native skills to buiild a list task management protoype mobile app , MyTaskApp which is a list task management app built using React Native based on a Expo GO ecosystem. This will need to utilise the creation of task screens with Create, Read, Update and Delete (CRUD) operations.

Functionalites:

1. Task List Screen When no tasks are available, it will display a message on the screen. Each task card shows the task title, description, due date, and its completion status. Tapping on the card toggles its completion state—turning the title to a strike-through with a status completed The card also has a: • Edit button • Delete icon with a confirmation alert At the bottom of the screen, there are two main buttons: • Add Task • Overall Summary which shows a popup summarising total number of tasks , how many completed and incompleted tasks with the result in percentage form. 2. Add Task Screen This screen allows users to create new tasks with these fields: • Title • Description • Due date • Additional details which are optional 3. Edit Task Screen Upon clicking the edit button in the card, user can update any existing task. The fields are pre-filled with the current information retrieved from any existing information written in the card . Behind the Scenes: Data Handling All tasks are stored in a central context called TaskContext. This ensures that adding, editing, deleting, and toggling tasks works from across all screens in the app. 4. There is a search bar, which filters tasks instantly based on title or description.
