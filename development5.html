Retro Game Quiz

Languages/Frameworks Used:
React Native
Expo Go

Objectives:

Application of React Native JS knowledge such as UseState, View, Text, StyleSheet, ScrollView, TouchableOpacity, Alert, Picker

Technical Competencies:


Mobile Application Development (React Native)

• Developed a mobile application using React Native
• Structured application using functional components

State Management & Logic

• Managed application state using React Hooks (useState)
• Implemented quiz scoring logic by comparing user input against predefined correct answers
• Calculated quiz results and performance percentage using programming logic

Props & Data Flow

• Passed data and event handler functions from parent components to child components using props
• Used callback functions to update parent state based on user interactions
• Implemented controlled components to ensure consistent handling of user-selected answers
• Enabled real-time updates of selected answers across the application

User Interface & Styling

• Designed responsive layouts using React Native StyleSheet
• Application of appropriate margin and padding, font size for improved readability
• Integrated icons to improve visual clarity
• Configured image scaling and layout using width and height styling
• Styled interactive elements such as buttons and pickers for better user experience

User Interaction & Feedback

• Implemented dropdown-based answer selection using Picker
• Provided immediate user feedback through Alert dialogs
• Feedback messages based on quiz performance

Media Handling

• Displayed different local image assets for each quiz question

Application Structure & Project Management Skills

• File Organisation into their respective folders for easier navigation.
• Applied proper coding conventions to components, variables, and styles
• Maintained a clear code structure by using beginner‑level writing and organising logic into modular parts.

Functionalites/Features:

Quiz Interface:
• Displays a series of quiz questions with accompanying images to provide visual context.

Answer Selection:
• Allows users to select answers using a dropdown picker for each question.
• Selected answers are stored and updated in real time as users interact and proceed to other questions.

Score Calculation:
• Compares user-selected answers against predefined correct answers upon submission.
• Calculates the total score and performance percentage automatically.
User Feedback:
Displays an alert message showing the quiz result including total correct answers with a feedback message based on performance level.

Visual Design:
Uses gradient styling, icons, images, and consistent layout to create a visually engaging and user-friendly interface.

Building process:

Mini project was part of a regular school assignment to practise React Native fundamentals through hands-on implementation.
LinearGradient was used for styling to make page more lively to user and make the quiz feel more engaging for users.
